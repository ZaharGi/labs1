Данный проект представляет собой лабораторную работу, посвященную освоению с Git и платформы GitHub. В рамках работы разработана программа для вычисления геометрических параметров трапеции. Программа реализует три основные функции: расчет периметра трапеции по четырем сторонам, вычисление площади через основания и высоту, а также нахождение длины средней линии по двум основаниям. Проект структурирован в виде репозитория, содержащего исполняемый код, файл README с описанием. Целью работы является закрепление навыков работы с Git, включая инициализацию репозитория, создание коммитов, работу с ветками и публикацию проекта на GitHub.
